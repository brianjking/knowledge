MacOS
=========

Show Dotfiles in Finder.app
--------------------------------
   
   ex. ::
   
	defaults write com.apple.Finder AppleShowAllFiles YES

* Reminder: Finder must be relaunched after running this command in Terminal/iTerm.app.
